# Number Guessing Game 
## Description
A simple python game where the computer generates a random number between 1 and 100 
and the player tries to guess it 

## Features
  -Random Number generations
  -User Input 
  -High and low hints
  -Attempt counter 
  -play again option

## Technologies
 -Python 3

## How to Run 
 ''' bash 
 python guessingnumber.py 
 '''
