# TO-DO LIST APP

## Description
A simple python-based To-Do List Application that helps users manage 
their daily tasks.Tasks are stored in a text file , so they remain
saved even after the program is closed.

## Features
 -view all tasks
 -add new tasks
 -delete the existing tasks
 -save tasks permanently using the file handling
 -simple menu-driven interface

## Technologies used
 -python
 -file handling
 -functions
 -loops
 -exception handling

## Project structure
 '''
    To-DO List App
     -TO-DO LIST-APP.py
     -tasks.txt
     -Readme3.md
'''

## How to Run 
''' bash
 python "TO-Do-List.py"
 '''
